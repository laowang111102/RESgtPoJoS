# 学生成绩管理系统 python112

学生成绩管理系统是一款基于 Flask 框架和 MySQL 数据库开发的 Web 应用程序。

## 技术栈

- 后端：Python 3.x
- Web 框架：Flask
- 数据库：MySQL

## 功能模块

### 学生模块

- 查看个人信息
- 修改个人信息
- 查看成绩

### 教师模块

- 查看基本信息
- 查看授课信息
- 查看学生选课信息
- 查看授课学生的成绩
- 成绩统计（直方图、饼图）
- 录入成绩
- 修改成绩
- 删除成绩

### 管理员模块

- 多项管理功能

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- Python 3.x
- Flask
- MySQL

## 部署步骤

请参考项目附带的部署文档进行环境搭建和项目部署。

## 目录结构

```
/
├── app             # Flask 应用目录
│   ├── models.py   # 数据模型
│   ├── views.py    # 视图逻辑
│   ├── templates   # HTML 模板
│   └── static      # 静态文件
├── venv            # 虚拟环境目录
├── requirements.txt  # 依赖列表
└── run.py          # 应用启动脚本
```

## 核心亮点

- 基于成熟的 Flask 框架，易于维护与扩展
- 清晰的三角色权限划分，功能模块分明
- 提供成绩统计图形化展示，直观了解数据分布
